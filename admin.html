Configura GitHub

Per publicar automàticament, necessites un token personal de GitHub. Es guarda només al teu navegador, mai s'envia enlloc més.

Crea'n un a github.com/settings/tokens amb scope repo. Caduca segons el que triïs.
Com obtenir el token? (detallat)
1. Obre aquest enllaç (ja ve preconfigurat)
2. A "Expiration", tria "No expiration" o la data que vulguis
3. Verifica que repo està marcat
4. Clic a "Generate token" al final
5. Copia el token (només es mostra un cop) i enganxa'l aquí a dalt
